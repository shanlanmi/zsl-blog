### 思路
每次点击，取消之前的 timeout，然后新设立 timeout

### 代码

```js
function clickEvent() {
  if timeoutId {
    window.clearTimeout(timeoutId);
  }
  var timeoutId
  timeoutId = window.setTimeout(function(){
    youCode();
    window.clearTimeout(timeoutId);
  }, 1000);
}
```
